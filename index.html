Configuration Needed

To use this app, you need to set up a free Firebase project:

  1. Go to the Firebase Console and create a new project.
  2. In your new project, go to Build > Firestore Database, create a database, and start it in Test mode.
  3. Go to Project Settings (⚙️) > Your apps, create a new Web app (</>).
  4. Copy the `firebaseConfig` object provided.
  5. Paste it into the HTML file where it says `// PASTE YOUR FIREBASE CONFIG HERE`.
